先看看哪种“二次元气场”最对你的胃口，后续再细化功能和动效。

🌸
Kawaii Card Grid

萌系糖果风

马卡龙配色、大圆角卡片、Q 版图标、弹性悬浮动画。整体像打开一盒彩色糖果，适合“今天吃啥”“B 站解析”这种轻松工具。

  • pastel 粉 / 薄荷绿 / 奶油黄
  • 圆角 20px+，阴影柔和
  • 图标带轻微弹跳入场
⚡
Cyberpunk Neon

赛博霓虹风

深色底、高饱和霓虹描边、故障艺术微动效、玻璃拟态卡片。适合“视频解析”这类带点黑客感的功能，夜晚使用很酷。

  • 深紫 / 黑蓝底 + 青/粉霓虹
  • 1px 发光边框 + 扫描线背景
  • 悬停时轻微 glitch / 光晕扩散
🎐
Minimal Moe

极简萌系风

留白充足、浅灰白底、细线图标、只有一个可爱的看板娘/吉祥物角落点缀。高级感与二次元不冲突，动画克制但极其顺滑。

  • 白 / 淡灰 + 单一强调色
  • 大量留白，卡片轻阴影
  • 滚动视差、磁吸按钮等微交互
